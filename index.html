Add Shadow Effect in CanvasJS Charts

This plugin allows you to add shadow effect to datapoints in CanvasJS Chart. Check out Github Repository for more info / examples.

Live Example

How to Use?

  • Enable shadow by setting shadowEnabled to true
  • Customize the color and positioning of shadow by setting color, offset & blur properties.
var chart = new CanvasJS.Chart("chartContainer", {
	.
	.
	.
	//Chart Options
	shadowEnabled: true,
	shadowStyle: {
		color: "black",
		offsetX: 4,
		blurLevel: 4
	}
	.
	.
	.
});
chart.render();
